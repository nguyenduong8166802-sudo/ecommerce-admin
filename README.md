# PRO Marketplace kiểu WinMart

Bản starter PRO gồm:
- Trang chủ kiểu siêu thị online
- Shop, chi tiết sản phẩm, giỏ hàng, checkout
- Admin dashboard
- Seller dashboard
- Tính hoa hồng theo seller
- CRUD sản phẩm cho admin và seller
- Lưu dữ liệu bằng file JSON để deploy nhanh

## Demo login
- Admin: `admin@duongmart.vn` / `admin123`
- Seller: `seller1@duongmart.vn` / `seller123`

## Chạy local
```bash
npm install
npm start
```

Mở: `http://localhost:10000`

## Deploy Render
- Build Command: `npm install`
- Start Command: `npm start`

## Lưu ý
Đây là bản starter PRO để đi nhanh. Muốn chạy thương mại thật nên nâng tiếp:
- upload ảnh file thật
- database MySQL/Postgres
- thanh toán online
- phân quyền chi tiết
- mã giảm giá
- vận chuyển
