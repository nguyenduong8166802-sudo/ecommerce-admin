# DUONG MART PRO FULL REDEPLOY ADMIN CMS

## Tính năng
- Auto create/update tables
- Admin CRUD sản phẩm full
- Seller CRUD sản phẩm full
- Marketing editable
- Flash Sale editable
- Banner quảng cáo editable
- Register/Login buyer-seller-admin
- Cart + checkout cơ bản

## Biến môi trường Render
- DATABASE_URL
- SESSION_SECRET
- PUBLIC_BASE_URL (khuyến nghị)

## Tài khoản mẫu
- admin@duongmart.vn / admin123
- seller1@duongmart.vn / seller123
- buyer1@duongmart.vn / buyer123

## Deploy
- Build command: npm install
- Start command: npm start
